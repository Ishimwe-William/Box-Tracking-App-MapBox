# Box Tracking App

This repository contains the Box Tracking App, a project designed to track and display box locations using MapBox and
Arduino.

## Integration with Arduino

This app works with Arduino code which can be accessed
here: [Arduino Sketch](https://github.com/Ishimwe-William/Box-tracking-with-esp32-cam)

## Features

- Real-time box location tracking
- Map visualization using MapBox
- Integration with Arduino for data retrieval

## Installation

1. Clone this repository:
   ```
   git clone https://github.com/Ishimwe-William/Box-Tracking-App-MapBox.git
   ```
2. Navigate to the project directory:
   ```
   cd box-tracking-app-mapbox
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the application:
   ```
   npm start
   ```

## Usage

- Make sure your Arduino device is set up and the sketch is uploaded.
- Open the app to monitor the real-time locations.
